# 7-Я

Статичный лендинг психологической практики Екатерины Юрьевны.

## Локальный просмотр

Откройте `index.html` в браузере или запустите любой статичный HTTP-сервер из корня проекта.

## Публикация

Проект не требует сборки: после загрузки в GitHub импортируйте репозиторий в Vercel. В настройках Vercel выберите Framework Preset: `Other`; build command и output directory оставьте пустыми.

Перед публикацией заменить плейсхолдеры/тексты юридической информации на проверенные юристом редакции.
